# Task 1 — Understanding AWS & Free Tier

## Goal

Demonstrate understanding of AWS basics and Free Tier usage by answering the following questions in your own words (3–4 lines each).

### Answers

#### Question 1 — What is an AWS account, and why do you need it at this stage?

An AWS Account gives a user access to a wide range of cloud infrastructure services like EC2 instances, Cloud front and the likes. It provides affordable access to computational services that would usually cost a huge sum upfront. At this stage of learning, the services AWS provides can be used to run small scale tests so as to undrstand the practical aspect of DevOp concepts

---

#### Question 2 — What is AWS Free Tier, and how long does it last?

AWS free tier is an account type available for new users on AWS. It provides the user with free credits over a period of 6 months and allows access to some low-level services.

---

#### Question 3 — Name three AWS Free Tier services and their free usage limits.

Amazon EC2 - 750 hours per month of t2.micro (or t3.micro in regions where t2.micro is unavailable) instance usage for 6 months
Amazon S3 - 5 GB of standard storage, 20,000 Get Requests, and 2,000 Put Requests for 6 months
Amazon DynamoDB - 25 GB of storage, 25 provisioned Write Capacity Units (WCU), and 25 provisioned Read Capacity Units (RCU) - Always Free
